# Webify

[English](../README.md) | [日本語](README.ja.md)

AIコーディングエージェントのための適応型ウェブリサーチ。ウェブを検索し、セマンティックグラフを構築し、統合された回答を取得します。ディープリサーチツールのわずか5%のコストで実現。

[GrapeRoot](https://graperoot.dev) によるスキル

## 機能概要

| ツール | 用途 | コスト |
|------|---------|------|
| `web_find(query)` | マルチソースウェブ検索 + 統合 | ~$0.003/クエリ |
| `web_lookup(url, query)` | 単一ページのグラフ検索 | ~$0.0005/クエリ |

**web_find** はDuckDuckGoで検索し、複数のソースからセマンティックグラフを並列で構築し、BM25を使って関連コンテンツを抽出し、Haikuで統合します。クエリの複雑さに応じて深度を適応的に調整します。単純な事実確認クエリは3ソースを参照し、多面的なリサーチクエリは6以上のソースにスケールし、マルチアスペクト検索を行います。

**web_lookup** は単一ページを取得し、見出し階層のグラフを構築し、関連ノードのみを返します（5,000〜50,000トークンの代わりに約250〜750トークン）。

## ベンチマーク

15件の未見クエリに対する、ClaudeのDeep Researchとのブラインド A/B評価。ジャッジ: Sonnet、正確性 + 網羅性 + 具体性をスコアリング（各1〜5点、最大15点/クエリ）。

| 指標 | Webify | Deep Research |
|--------|--------|--------------|
| 品質スコア | **68/75** (90.7%) | 73/75 (97.3%) |
| クエリあたりコスト | **~$0.003** | ~$0.05+ |
| レイテンシ | **30〜90秒** | 80〜280秒 |
| コスト効率 | **18倍優れている** | ベースライン |

WebifyはDeep Researchの品質の91%を、コスト5%で達成します。

## 設定

| 環境変数 | 必須 | 説明 |
|---------|----------|-------------|
| `ANTHROPIC_API_KEY` | `web_find` に必要 | Haiku統合 + バンディット学習 |
| `BRAVE_SEARCH_API_KEY` | 推奨 | 安定した検索（無料で月2,000クエリ） |
| `WEBIFY_CACHE_DIR` | いいえ | キャッシュの保存場所（デフォルト: `~/.cache/webify`） |
