# Webify

[English](../README.md) | [Русский](README.ru.md)

Адаптивный веб-поиск для AI-агентов программирования. Ищите в интернете, стройте семантические графы, получайте синтезированные ответы — за 5% стоимости инструментов глубокого исследования.

Навык от [GrapeRoot](https://graperoot.dev)

## Что он делает

| Инструмент | Назначение | Стоимость |
|------------|-----------|-----------|
| `web_find(query)` | Мультиисточниковый веб-поиск + синтез | ~$0.003/запрос |
| `web_lookup(url, query)` | Извлечение графа с одной страницы | ~$0.0005/запрос |

**web_find** выполняет поиск через DuckDuckGo, параллельно строит семантические графы из нескольких источников, извлекает релевантный контент с помощью BM25 и синтезирует ответ через Haiku. Глубина поиска адаптируется в зависимости от сложности запроса — простые фактологические запросы обращаются к 3 источникам, многомерные исследовательские запросы масштабируются до 6+ источников с многоаспектным извлечением.

**web_lookup** загружает одну страницу, строит граф на основе иерархии заголовков и возвращает только релевантные узлы (~250-750 токенов вместо 5 000-50 000).

## Конфигурация

| Переменная окружения | Обязательна | Описание |
|---------------------|-------------|----------|
| `ANTHROPIC_API_KEY` | Для `web_find` | Синтез через Haiku + обучение бандита |
| `BRAVE_SEARCH_API_KEY` | Рекомендуется | Надёжный поиск (бесплатно 2 тыс. запросов/мес) |
| `WEBIFY_CACHE_DIR` | Нет | Расположение кэша (по умолчанию: `~/.cache/webify`) |
